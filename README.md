Proyecto desarrollado para practicar pruebas unitarias y aplicar buenas prácticas de testing en aplicaciones Java.

##  Características

* Pruebas unitarias con JUnit.
* Uso de Mockito para simulación de dependencias.
* Cobertura superior al 70%.
* Validación de lógica de negocio.

##  Tecnologías

* Java
* JUnit 5
* Mockito
* Maven
* Git & GitHub

---

##  Estructura

```text
src
├── main
└── test
    ├── service
    ├── repository
    └── controller
```

##  Instalación

```bash
git clone https://github.com/TU_USUARIO/pruebas-unitarias.git
cd pruebas-unitarias
mvn test
```

##  Métricas

* Más de 20 pruebas unitarias.
* Cobertura superior al 70%.
* Uso de Mockito para pruebas aisladas.
